# 00 硬件总览：先认识你手里的 ESP32-S3

这一章先解决最容易卡住的几个问题：开发板到底是哪一块、USB 口怎么用、BOOT/RESET 是什么、外设要怎么插到面包板上。

## 开发板、模组、芯片的区别

你手上的一般是“开发板”，开发板中间那块金属屏蔽罩或小板通常是“模组”，模组里面才是 ESP32-S3 芯片。

```text
开发板
  ├─ USB-C / Micro-USB 口：供电、烧录、串口日志
  ├─ 稳压芯片：把 USB 5V 转成 3.3V
  ├─ BOOT / RESET 按键：进入下载模式或复位
  ├─ 排针：把 GPIO、3V3、5V、GND 引出来
  └─ ESP32-S3 模组：芯片、Flash，有些还带 PSRAM
```

本教程默认你使用通用 ESP32-S3 开发板。ESP32-S3-DevKitC-1、ESP32-S3-N8R8、ESP32-S3-N16R8 这类板子都可以。只要你的板子能跑 ESP-IDF，且有可用 GPIO，就能跟着做。

## USB 口从哪里插

很多 ESP32-S3 开发板只有一个 USB 口；也有些板子有两个 USB 口。

常见标识：

| 板上标识 | 通常含义 | 新手建议 |
| --- | --- | --- |
| `USB` / `UART` / `COM` / `PROG` | 通过 USB-UART 芯片连接串口 | 优先用于烧录和看日志 |
| `USB-OTG` / `OTG` / `Native USB` | ESP32-S3 原生 USB，可能用于 USB Serial/JTAG 或设备模式 | 能用，但不同板子配置差异更大 |
| `5V` / `VIN` | 供电针脚，不是 USB 口 | 不要和电脑 USB 口混淆 |

如果你不确定，用这个判断顺序：

1. 插上数据线。
2. 电脑出现新的串口设备。
3. `idf.py flash monitor` 能看到日志。
4. 这个口就可以作为教程主 USB 口。

如果插上只亮灯但电脑没有新串口，优先怀疑 USB 线是充电线。

## BOOT 和 RESET 按键

开发板上常见两个按键：

| 按键 | 作用 |
| --- | --- |
| `RESET` / `RST` / `EN` | 复位芯片，程序重新运行 |
| `BOOT` / `IO0` | 启动模式选择，也常接到 GPIO0 |

ESP32-S3 复位时会读取一些特殊引脚的电平来决定启动模式。GPIO0 是最常见的下载模式相关引脚：

```text
正常启动：松开 BOOT，按一下 RESET
手动下载：按住 BOOT，点一下 RESET，开始烧录后松开 BOOT
```

很多开发板有自动下载电路，`idf.py flash` 时不需要手动按键。如果烧录卡在连接阶段，再用手动下载流程。

## 面包板怎么接

基础外设推荐用面包板连接。

```text
ESP32-S3 3V3  -> 面包板红色电源轨
ESP32-S3 GND  -> 面包板蓝色地线轨
GPIO          -> 外设信号脚
```

注意：

- 红色电源轨不是天然 3.3V，必须由开发板 `3V3` 接过去。
- 蓝色地线轨不是天然 GND，必须由开发板 `GND` 接过去。
- 有些面包板左右两边电源轨中间断开，需要用跳线桥接。
- 接线前断开 USB，接好再上电。

## 最小器材清单

基础实验：

- ESP32-S3 开发板
- USB 数据线
- 面包板
- 公对公杜邦线
- LED
- 220 欧到 1k 欧电阻
- 按键
- 10k 电位器

语音实验：

- INMP441 I2S 麦克风
- MAX98357A I2S 功放
- 4 欧或 8 欧小扬声器
- 额外 5V 供电可选；功放大音量时更稳

## 从外观看针脚

开发板排针上的文字是“丝印”，例如 `GND`、`3V3`、`5V`、`GPIO4`、`IO4`、`RX`、`TX`。不同板子的丝印风格不同。

```text
IO4、GPIO4、G4 通常都表示 GPIO4
3V3 表示 3.3V 输出
5V / VIN 通常来自 USB 5V 或外部输入
GND 表示地
```

不要只凭网上另一块板子的照片接线。ESP32-S3 开发板很多，针脚位置可能完全不同。接线时以你自己开发板的丝印和原理图为准。

## 本教程的默认外设连接

```text
LED:
  GPIO2 -> 电阻 -> LED 正极
  LED 负极 -> GND

按键:
  GPIO0 -> 按键 -> GND
  程序启用内部上拉

电位器:
  中间脚 -> GPIO4
  两侧脚 -> 3V3 / GND

INMP441:
  VCC -> 3V3
  GND -> GND
  SCK -> GPIO5
  WS  -> GPIO6
  SD  -> GPIO7
  L/R -> GND

MAX98357A:
  VIN  -> 5V 或 3V3
  GND  -> GND
  BCLK -> GPIO15
  LRC  -> GPIO16
  DIN  -> GPIO17
  SPK+ / SPK- -> 扬声器
```

如果你的板子某个 GPIO 被占用，换一个普通空闲 GPIO，并修改对应示例 `main.c` 顶部的宏。
