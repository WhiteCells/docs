### find 文件查找

指定目录下查找指定后缀的文件：

```sh
find ./ \( -name "*.txt" -o -name "*.pdf" \) -print
```

正则方式查找指定后缀的文件：

```sh
find ./ -regex ".*\(\.txt|\.pdf\)$"
# -iregex 忽略大小写的正则
```

指定目录下查找非指定后缀的文件：

```sh
find ./ ! -name "*.txt" -print
```



### grep 文件搜索

```sh
grep match_patten file
```

常用参数：

- -o 只输出匹配的文本行
- -v 只输出没有匹配的文本行
